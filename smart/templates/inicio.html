Smart-Pulse - Guía de Uso

1. Configuración Inicial

REGISTRO: Completa tu perfil personal en la sección "Registro" para obtener mediciones precisas de calorías y zonas de frecuencia cardíaca.

  • Ingresa tu nombre completo, fecha de nacimiento y género
  • Proporciona tu altura y peso actual
  • El sistema calculará automáticamente tu edad e IMC

2. Monitoreo en Tiempo Real

SESIÓN: Inicia una sesión de monitoreo cardíaco conectando tu dispositivo sensor.

  • Asegúrate de que el dispositivo ESP32 esté conectado a la red WiFi
  • Verifica que el sensor esté enviando datos al puerto UDP 8888
  • El sistema mostrará el BPM en tiempo real y calculará las calorías
  • Puedes ver tu zona de frecuencia cardíaca durante el ejercicio

3. Seguimiento de Progreso

HISTORIAL: Revisa todas tus sesiones anteriores y analiza tu progreso.

  • Ve estadísticas completas de cada sesión
  • Analiza tendencias de BPM promedio, mínimo y máximo
  • Revisa las calorías quemadas en cada entrenamiento
  • Busca sesiones por fecha, usuario o género

4. Solución de Problemas

Conexión del Sensor:

  • Verifica que el ESP32 esté encendido y conectado al WiFi
  • Asegúrate de que el sensor de pulso esté correctamente conectado
  • El indicador debe mostrar "Conectado" cuando funcione correctamente

Datos:

  • Los datos se guardan automáticamente en archivos CSV
  • Si hay problemas de conexión, reinicia la aplicación
  • Verifica la configuración de red si no hay comunicación

5. Información Técnica

Especificaciones:

  • Puerto UDP: 8888 (configurable en RecibePulso.py)
  • Rango de BPM válido: 20-220 latidos por minuto
  • Cálculo de calorías: Fórmula de Keytel (precisa para ejercicio)
  • Almacenamiento: Archivos CSV por fecha en carpeta 'historiales/'
